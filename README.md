# STEP-SENSE-Blind-Mobility-Using-IoT

📌 Overview

STEP SENSE is an IoT-based assistive system designed to help visually impaired individuals navigate safely and independently. The system uses sensors to detect obstacles and water hazards in real time and provides alerts through sound and vibration. It also supports mobile connectivity for monitoring and enhances efficiency using energy generation.

🎯 Objectives
To improve mobility and safety for visually impaired users
To detect obstacles using ultrasonic sensors
To identify wet or slippery surfaces using moisture sensors
To provide real-time alerts through buzzer and vibration
To enable communication with a mobile application using IoT
⚙️ System Components
🔹 Hardware
Arduino UNO (Microcontroller)
Ultrasonic Sensor (Obstacle Detection)
Moisture Sensor (Water Detection)
Buzzer (Sound Alert)
Vibration Motor (Haptic Feedback)
ESP8266 Wi-Fi Module (Communication)
Piezoelectric Sensor (Energy Generation)
Power Supply
🔹 Software
Arduino IDE
Embedded C / Arduino Programming
Mobile Application (for monitoring)
🔄 Working Principle
Sensors continuously collect data from the surroundings
Ultrasonic sensor detects obstacles
Moisture sensor detects water or wet surfaces
Microcontroller processes the data
Alerts are generated using buzzer and vibration motor
Data is sent to mobile app using Wi-Fi module
Piezoelectric sensor generates power during movement
